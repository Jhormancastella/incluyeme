## ✨ Características

- Página de inicio con hero, estadísticas y previews dinámicos
- Galería de imágenes con filtros, lightbox y paginación
- Galería de videos con soporte YouTube, Vimeo y Facebook
- Sección de proyectos y recursos municipales
- Formulario de registro comunitario
- Página de historia de la discapacidad en Colombia
- Panel de administración completo (CRUD de todo el contenido)
- Modo claro/oscuro con persistencia
- Diseño responsive mobile-first
- Accesibilidad: navegación por teclado, roles ARIA, skip link

## 🚀 Uso local

1. Clonar el repositorio
2. Abrir `index.html` en un navegador moderno
3. No requiere servidor ni dependencias

> Requiere navegador con soporte ES6 Modules (Chrome 87+, Firefox 78+, Safari 14+)
